# 3 年級電腦課程

## 課程內容
- 作業系統操作開關機
- 電腦軟硬體
- 滑鼠基本操作 (左右鍵、滾輪；點一下、快點兩下、拖曳)
- 個人化設定
- 像素概念 (像素繪圖、七段顯示器)
- 英文、數字、符號輸入
- 台南市 OpenID 申請
- E-game 挑戰
- Code.org 基礎課程
- 中文輸入法、聊天室
- 選取操作 (連續選取、挑選)
- 檔案管理操作 (複製、移動)
- 檔案結構概念 (資料夾樹狀結構)

## 遊戲

- 填色遊戲 (四色問題)
- 立體解鎖
- 找壽司解謎遊戲
- 數學遊戲
